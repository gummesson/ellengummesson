## Git

Aliases for common [Git](http://www.git-scm.com/) tasks.

| Alias    | Command                 |
|----------|-------------------------|
| `g`      | `git`                   |
| `gst`    | `git status`            |
| `gd`     | `git diff`              |
| `gl`     | `git pull`              |
| `gp`     | `git push`              |
| `gcmsg`  | `git commit -m`         |
| `gco`    | `git checkout`          |
| `gr`     | `git remote`            |
| `gb`     | `git branch`            |
| `ga`     | `git add`               |
| `gm`     | `git merge`             |
| `ggpull` | Pull current[^1] branch |
| `ggpush` | Push current branch     |

[^1]: you can see the current branch right on the command line, depending on your `Zsh` theme.
